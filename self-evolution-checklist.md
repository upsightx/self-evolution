# 自我进化检查清单

## 每次任务后必做

- [ ] 子 Agent 产出用 Critic 评分（80 分及格）
- [ ] <80 分打回重做
- [ ] 重要产出我自己复核
- [ ] 记录决策到 memory_db（包含 rejected_alternatives）

## 每周审计

- [ ] 回顾 memory/*.md，提取教训
- [ ] 检查 agent-stats.json 成功率
- [ ] 审视子 Agent 指令模板有效性
- [ ] 更新 MEMORY.md

## 每月复盘

- [ ] 记忆压缩（30天前的内容）
- [ ] 清理失效的 skill/工具
- [ ] 评估新工具是否真正有用

## 原则

1. **不造没验证过的工具** — 先 small test，再部署
2. **指令精简** — 关键约束放第一段，不超过 20 行
3. **可执行过滤** — 不用模糊指标，用关键词匹配
4. **自己监督自己** — 子 Agent 是手，我是大脑
