# 量子位 — 差异配置

**源标识**: `qbitai`
**源名称**: 量子位
**特有指标**: 🇨🇳 国内动态

## 数据源与抓取

1. 用 `web_search` 搜索量子位最新文章：
   - "site:qbitai.com AI" 最近 3 天
   - "site:qbitai.com 大模型" 最近 3 天
   - "site:qbitai.com Agent" 最近 3 天

2. 用 `web_fetch` 抓取量子位首页：
   - `https://www.qbitai.com/`

3. 用 `web_search` 补充机器之心：
   - "site:jiqizhixin.com AI" 最近 3 天

4. 对高价值文章（分数≥7），用 `web_fetch` 读取全文提取核心信息

## 提取字段

标题、URL、来源（量子位/机器之心）、发布日期、核心内容摘要

## 特有标注
- 🇨🇳 国内公司/团队
- 🏆 国内首发/独家

## 输出格式示例

```
### 1. [标题](url) | 分数: 8.0 | 🇨🇳 国内动态
- **摘要**: 核心内容（2-3句）
- **标签**: [AI] [大模型] ...
- **来源**: 量子位
- **为什么值得记**: 价值说明
```
